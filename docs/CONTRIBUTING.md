# 🤝 Contributing to Behavior School

Thank you for your interest in contributing to **Behavior School**! We welcome community input, scientific paper recommendations, web platform enhancements, and video topic suggestions.

---

## 🎯 How You Can Contribute

### 1. Suggest Video Topics & Questions
Have a question about human behavior, focus, procrastination, or relationships that you want explained?
- Open an Issue with the tag `video-idea`.
- Outline the core question, why it matters, and any relevant research papers.

### 2. Submit Scientific Studies or Research
If you find a high-impact paper in psychology or neuroscience that provides a practical behavioral framework:
- Submit a pull request or issue adding the paper to our [CURRICULUM.md](CURRICULUM.md) or mental models library.

### 3. Enhance the Web Platform (`behavior-school.github.io`)
We welcome improvements to the Next.js web application:
- Bug fixes or accessibility enhancements.
- Adding new mental models or cognitive biases to `MentalModelsLibrary.tsx`.
- Improving interactive tools like the `BehaviorDiagnostic.tsx`.

---

## 💻 Local Development & PR Workflow

1. **Fork & Clone**:
   ```bash
   git clone https://github.com/YOUR-USERNAME/behavior-school.github.io.git
   cd behavior-school.github.io
   ```

2. **Install Dependencies**:
   ```bash
   bun install
   ```

3. **Make Your Changes**:
   - Ensure clean TypeScript code and Tailwind styling.
   - Run `bun run build` locally to verify static export builds without errors.

4. **Submit a Pull Request**:
   - Push your branch to GitHub and create a Pull Request with a summary of changes.

---

## 📜 Code of Conduct & Content Standards

- **Scientific Integrity**: All psychology and neuroscience content must be research-backed.
- **Empathetic & Constructive**: Content should empower people to change behavior without judgment.
- **Clear & Concise**: Avoid academic jargon when plain English conveys the same concept.
