## 📌 About Behavior School

**Behavior School** is an educational media brand and web platform dedicated to bridging the gap between scientific research (psychology, neuroscience, cognitive science) and everyday human behavior.

Instead of teaching psychology as an academic textbook subject full of dry jargon, Behavior School focuses on **practical psychology**—actionable knowledge that can be immediately applied to daily decisions, habit formation, focus, and emotional regulation.

---

## 🎯 Mission & Vision

* **Mission**: Help millions of people understand their own minds and change their behavior through simple, highly visual, research-backed education.
* **Vision**: Become the world's most trusted, 100% free educational platform for practical psychology, neuroscience, and human behavior.
* **Content Philosophy**: Knowledge should lead to action. No textbook fluff. No boring lectures. Every video and lesson provides clear, actionable takeaways.

## 🏛️ The 6 Core Pillars

Behavior School organizes behavioral science into 6 practical categories:

| Pillar | Focus & Key Topics | Core Takeaway |
| :--- | :--- | :--- |
| **1. Psychology Explained** | Cognitive Dissonance, Confirmation Bias, Halo Effect, Zeigarnik Effect, Dunning-Kruger | Recognize cognitive traps biasing your daily perception. |
| **2. Neuroscience & Brain** | Dopamine loops, Neuroplasticity, Memory encoding, Stress/Cortisol, Sleep optimization | Work with your brain chemistry instead of fighting against it. |
| **3. Habits & Discipline** | Habit loops, Breaking bad habits, Motivation vs Discipline, Environment design | Systems outweigh willpower. Design environments that enforce consistency. |
| **4. Learning & Productivity** | Active Recall, Spaced Repetition, Deep Work protocols, Flow states, Time management | Master cognitive efficiency to learn complex skills in record time. |
| **5. Social Psychology** | Persuasion science, Attraction, Social confidence, First impressions, Relationships | Build strong social intuition and navigate human interactions with clarity. |
| **6. Mental Models** | First Principles Thinking, Second Order Thinking, Pareto Principle (80/20), Inversion | Filter out noise and solve hard problems by targeting root causes. |

## 🛠️ Web Platform Features

The web application built at **[behavior-school.github.io](https://behavior-school.github.io/)** includes:

* **Interactive "Why Do I Do This?" Mind Diagnostic**: Visitors select their current struggle (Procrastination, Low Focus, Overthinking, Inconsistency, Social Anxiety) to get an instant neuro-psychological breakdown and 3-step action protocol.
* **Questions Explorer**: Searchable accordion breakdown covering fundamental questions like *"Why do we procrastinate?"* and *"Why do emotions overpower logic?"*
* **Mental Models & Cognitive Biases Library**: Searchable database of timeless thinking models with real-life application guides.
* **Featured Video Series Spotlight**: Showcasing the *Real Self Series* (*You Are Not Lazy*, *How to Learn Faster*, *Stop Overthinking*) and *Educational Deep Dives*.

## 💻 Tech Stack & Architecture

* **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)

## 🚀 Local Development Setup

### Prerequisites
- [Node.js v20+](https://nodejs.org/)
- [Bun](https://bun.sh/)

### Steps

1. **Clone the repository**:
   ```bash
   git clone https://github.com/behavior-school/behavior-school.github.io.git
   cd behavior-school.github.io
   ```

2. **Install dependencies**:
   ```bash
   bun install
   ```

3. **Start the development server**:
   ```bash
   bun run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.

4. **Test static build & export locally**:
   ```bash
   bun run build
   ```
